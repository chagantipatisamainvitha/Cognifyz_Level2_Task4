# Cognifyz Internship – Level 2 Task 4  
## Role: Data Visualization Intern

## Task Title
Duration and Expectations Analysis in Excel

## Objective
To analyze investment durations mentioned by participants and their expectations from investments using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Column Chart
- Radar Chart

## Analysis Performed
- Summarized investment duration data using pivot tables.
- Visualized the distribution of investment duration using a column chart.
- Summarized participants' investment expectations using pivot tables.
- Visualized investment expectations using a radar chart.

## Outcome
The analysis provides insights into participants’ preferred investment durations and the key expectations they have from their investments.
